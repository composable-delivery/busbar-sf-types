## Version Tracking

Each version of this crate corresponds to a specific version of `@salesforce/types`. The crate version is aligned with the npm package version:

- `@salesforce/types@1.0.0` → crate `1.0.0`
- `@salesforce/types@1.1.0` → crate `1.1.0`
- `@salesforce/types@1.1.0` (with codegen fixes) → crate `1.1.1` (patch)

You can find which @salesforce/types version generated your types by:
1. Checking the crate version in your Cargo.toml
2. Looking at the git commit that introduced the version
3. Checking `.typegen-reports/salesforce_types_version.txt` in this repository
